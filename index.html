Do This

Do This is a web application I built to help manage and achieve personal goals. I made it for practise, learning and because I wanted to use it.

  • Define your goals
  • Define actions (steps to take in order to work towards those goals)
  • Order your goals to prioritise them
  • Be shown a list of prioritised actions you should do based on the goals you have set
Uses Angular for the front end and Firebase for the backend. Created Feb 2016.

Interval Timer

An Android app I built for quick, easy and customisable interval timers.

  • Setup timers to go off at specific intervals. Hide or show time remaining.
  • Set up a list of timers to run in a sequence or be selected from randomly.
  • Optionally define a range from which your random interval durations may be selected.
  • Save your timers for later so you don't have to set them up again every day.
